Conceito

Um algoritmo é uma sequência finita e bem definida de instruções ou passos que resolvem um problema ou executam uma tarefa específica.

Estruturas

Sintaxe

Palavras-Chave

As palavras-chave são reservadas e têm um significado específico para o compilador.

Instruções

Instruções são os comandos que indicam ao computador o que fazer, como imprimir valores ou executar cálculos.

Blocos de Código

Blocos de código são agrupamentos de instruções que são executadas juntas.

Comentários

Comentários são usados para documentar o código e não são executados pelo programa.

Comandos de Decisão e Repetição

Variáveis e Tipos de Dados

As variáveis são usadas para armazenar dados e podem ser de diferentes tipos: inteiros, strings, booleanos, etc.

Atribuição de Valores

A atribuição de valores em variáveis é feita com o operador "=".

Operadores

  • Operadores Aritméticos: Somar, subtrair, multiplicar, dividir, etc.
  • Operadores Relacionais: Comparar valores (>, <, ==, etc.).
  • Operadores Lógicos: Trabalham com expressões booleanas (&&, ||, !).
  • Operadores de Atribuição: Atribuem valores a variáveis (+=, -=, etc.).

Comandos de Decisão

Estrutura de controle para executar diferentes blocos de código, dependendo de condições:

  • Operador Ternário
  • Estrutura if-else
  • Estrutura if-elseif-else
  • Comandos de Decisão Aninhados

Comandos de Repetição

Estruturas que permitem a execução repetida de blocos de código:

  • Laço for
  • Laço while
  • Laço do...while
  • Controle de laço: break e continue
  • Comandos de Repetição Aninhados
